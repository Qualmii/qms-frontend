## 🎓 Для Разработчиков

### Структура проекта
1. **Components** - переиспользуемые UI части
2. **Services** - интеграция с API и WebSocket
3. **Stores** - управление состоянием (Pinia)
4. **Views** - полные страницы приложения
5. **Types** - TypeScript интерфейсы

### Добавление новой функции
1. Создать тип в `types/api.ts`
2. Добавить метод в `services/api.ts`
3. Добавить действие в соответствующий `store`
4. Создать компонент в `components/`
5. Добавить маршрут в `router/index.ts`

## 📞 Техническая Поддержка

**Проблема:** Приложение не стартует  
**Решение:**
```bash
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
npm run dev
```

**Проблема:** WebSocket не подключается  
**Решение:** Проверить VITE_WS_URL в .env.local

**Проблема:** TypeError при компиляции  
**Решение:** Очистить браузер кэш и перезагрузить
