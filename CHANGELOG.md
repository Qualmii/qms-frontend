## 🆕 НОВЫЕ СЕРВИСЫ (2)

### 1. `src/services/websocket.ts`
**Тип:** TypeScript сервис  
**Строк:** 80+  
**Описание:** WebSocket/Socket.io интеграция  
**Класс:** WebSocketService (singleton)  
**Методы:**
- `connect(token)` - подключиться к WebSocket серверу
- `disconnect()` - отключиться
- `on(event, callback)` - подписаться на событие
- `off(event, callback)` - отписаться от события
- `send(event, data, callback)` - отправить событие
- `emit(event, ...args)` - вызвать локальные слушатели
- `isConnected()` - проверить статус подключения

**События (15+):**
- `onMessageReceived` - получение сообщения
- `onMessagesRead` - сообщение прочитано
- `onChatCreated` - чат создан
- `onChatUpdated` - чат обновлен
- `onChatDeleted` - чат удален
- `onUserJoinedChat` - пользователь присоединился
- `onUserLeftChat` - пользователь ушел
- `onIncomingCall` - входящий звонок
- `onCallAnswered` - звонок принят
- `onCallRejected` - звонок отклонен
- `onCallEnded` - звонок завершен
- `onICECandidate` - ICE кандидат получен
- `onSDPAnswer` - SDP ответ получен
- `onUserOnline` - пользователь онлайн
- `onUserOffline` - пользователь офлайн
- И еще 3+

**Конфигурация:**
- Автоматическое переподключение
- JWT токен в заголовке auth
- Обработка различных событий

### 2. `src/services/webrtc.ts`
**Тип:** TypeScript сервис  
**Строк:** 184  
**Описание:** WebRTC для видео/аудио звонков

**Методы:**
- `initializeMedia(constraints)` - получить медиапоток
- `stopLocalStream()` - остановить локальный поток
- `createPeerConnection()` - создать peer соединение
- `addICECandidate()` - добавить ICE кандидат
- `processPendingCandidates()` - обработать очередь кандидатов
- `setRemoteDescription()` - установить удаленное описание
- `getLocalStream()` - получить локальный поток
- `getRemoteStream()` - получить удаленный поток
- `getCurrentCall()` - получить текущий звонок
- `setCurrentCall()` - установить текущий звонок
- `endCall()` - завершить звонок
- `toggleAudio()` - включить/отключить звук
- `toggleVideo()` - включить/отключить видео
- `getCallDuration()` - получить длительность звонка

**Конфигурация:**
- STUN серверы Google
- Trickle ICE
- Поддержка видео (1280x720)
- Обработка ошибок
